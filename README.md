## FAQ

**1) Is this a VPN or a proxy?**  
It supports a VPN tunnel workflow (encrypted) and proxy-style routing patterns. Use VPN mode for Netflix for consistent DNS and encryption.

**2) Does it keep logs?**  
It’s built around a **No-Logs policy** posture. Keep your own device clean too (browser and OS caches are still yours).

**3) What if Netflix detects it?**  
Switch regions/servers, reconnect, and re-check DNS/IP leaks. Detection is usually a DNS mismatch or a flagged exit IP.

**4) Will it slow my connection down?**  
Any VPN adds overhead, but the default profiles are tuned for **high speed** streaming and a **stable connection**.
